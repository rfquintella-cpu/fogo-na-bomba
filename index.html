📲 Add the Calendar to Your Phone

All match times automatically show in your local timezone. Follow the steps for your device:

🍎

iPhone / iPad

  1. Tap Download Full Calendar above
  2. When Safari asks, tap "Open in Calendar"
  3. Tap Add All — done! ✅

Times auto-convert to your local time.

🤖

Android / Google Calendar

  1. Download the .ics file
  2. Open Google Calendar → Settings → Import
  3. Select the file → choose your calendar → Import

Or: open the file and your phone may offer to import it directly.

💼

Outlook / Windows

  1. Download the .ics file
  2. Double-click the file → Outlook opens it
  3. Click Import → done!

Works with Outlook.com and desktop Outlook.

💡 Pro tip: Each match card on the schedule also has its own Add to Calendar button — perfect if you only want specific games (like Brazil's matches! 🇧🇷).
